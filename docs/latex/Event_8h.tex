Configures how \mbox{\hyperlink{classEvent}{Event}} should be shown, when written to stream with \texorpdfstring{$<$}{<}\texorpdfstring{$<$}{<} operator. 


\begin{DoxyParams}{Parameters}
{\em setting} & A number, defining the different options. It\textquotesingle{}s best if it is passed as binary number.\\
\hline
\end{DoxyParams}
Sets a configuration setting for how operator\texorpdfstring{$<$}{<}\texorpdfstring{$<$}{<} should behave. Setting is stored as a global variable in the cpp file, so it\textquotesingle{}s independent of \mbox{\hyperlink{classEvent}{Event}} instance.

Each bit in the number is a certain setting. Each bit from left to right represents one of these options, from top to bottom\+:
\begin{DoxyEnumerate}
\item Write count of bought tickets (available only if 5. is set)
\item Write count of reserved tickets (available only if 5. is set)
\item Write reservations
\item Write bought tickets
\item Write \mbox{\hyperlink{classHall}{Hall}} information
\end{DoxyEnumerate}
